The heights module contains both a five-step height scale based on powers of two as well as a series of percentage values. Explicit values can be used inside of any parent. Percentage values will only work inside of a parent element that has a declared height.

Class name structure is as follows: 

Base:
  h = height

Modifiers
  1 = 1st step in height scale
  2 = 2nd step in height scale
  3 = 3rd step in height scale
  4 = 4th step in height scale
  5 = 5th step in height scale

  -25
  -50
  -75
  -100

  -auto  = string value auto

Media Query Extensions:
  -ns = not-small
  -m  = medium
  -l  = large

@custom-media --breakpoint-not-small screen and (min-width: 48em);
@custom-media --breakpoint-medium screen and (min-width: 48em) and (max-width: 64em);
@custom-media --breakpoint-large screen and (min-width: 64em);

/*

   HEIGHTS

*/

/* Height Scale */

.h1 { height: 1rem; }
.h2 { height: 2rem; }
.h3 { height: 4rem; }
.h4 { height: 8rem; }
.h5 { height: 16rem; }

/* Height Percentages */

.h-25 {  height:  25%; }
.h-50 {  height:  50%; }
.h-75 {  height:  75%; }
.h-100 { height: 100%; }

/* String Properties */

.h-auto {     height: auto; }
.h-inherit {  height: inherit; }
