Single purpose classes to set the font-style of any element at any breakpoint.

Italics can be used to empshasie on a piece of content. Some common uses of italics include: titles, vehicle names, letters of the alphabet, scientific terms, and quotes.

Examples

<p class="i"></p>

Readers want what is important to be clearly laid out; they will not read what is too troublesome.

<p class="fs-normal"></p>

A quote from Jan Tschichold about typography.

MDN - Font Weight
src/_font-style.css 

@custom-media --breakpoint-not-small screen and (min-width: 48em);
@custom-media --breakpoint-medium screen and (min-width: 48em) and (max-width: 64em);
@custom-media --breakpoint-large screen and (min-width: 64em);

/*

   FONT STYLE

*/

.i         { font-style: italic; }
.fs-normal { font-style: normal; }

@media (--breakpoint-not-small) {
  .i-ns       { font-style: italic; }
  .fs-normal-ns     { font-style: normal; }
}

@media (--breakpoint-medium) {
  .i-m       { font-style: italic; }
  .fs-normal-m     { font-style: normal; }
}

@media (--breakpoint-large) {
  .i-l       { font-style: italic; }
  .fs-normal-l     { font-style: normal; }
}
